## 执行规则

- App 内部 UI 全部来自真实截图。
- 外层包装只做标题、卖点、背景、设备框和品牌区。
- 不重画、不替换、不脑补 App 页面内容。
- 尺寸统一为 `1290 x 2796`，方便移动端应用市场纵向素材预览。
- 当前版本用于测试方向和快速挑选，不作为最终干净上架数据锁稿。

## 运营角度

1. 首页图强调“一屏掌控”，适合作为第一张总卖点图。
2. 实时数据图强调“看见进度”，适合承接效率复盘。
3. 四象限图强调“优先级”，适合说明任务规划方法。
4. 日程图强调“不遗漏”，适合说明时间安排。
5. 我的页图强调“工具箱”，适合做功能集合或个人中心介绍。

## 上架前建议

- 使用更干净的演示账号数据重新采集截图，避免测试任务标题、空数据状态和 debug 溢出提示进入最终版本。
- 最终锁稿前可从这 5 张里选择 3-5 张，再逐张强化文案和局部构图。
