## 推荐链路

录音转文字不要直接把本地文件传给百炼。百炼录音文件识别需要可访问的音频 URL，因此推荐链路是：

1. Flutter 客户端本地录音，生成临时音频文件。
2. 停止录音后拿到本地路径、时长、文件大小。
3. 通过 `AliyunStoreManager.getInstance().uploadFileByFilePath(...)` 上传到 OSS。
4. 上传时必须传 `DocType.audio`，并尽量传真实音频扩展名。
5. 拿到 OSS 公网 URL 后，调用百炼 Paraformer 录音文件识别接口。
6. 百炼接口异步返回 `task_id`，轮询任务状态。
7. 任务成功后下载 `transcription_url`，解析最终整句文本。
8. 将文本回填给业务方法，例如 AppAIPlugin 输入框。

## AppAIPlugin 内联录音方式

AppAIPlugin 是 HTML/Web GUI，录音功能不要弹 Flutter 弹窗。推荐让 Web 侧只负责 UI 状态和按钮事件，真正录音由 Flutter 宿主侧工具类完成。

当前消息约定：

```ts
await ideMessenger.request("appAi/startVoiceRecording" as any, undefined as any);
await ideMessenger.request("appAi/stopVoiceRecording" as any, undefined as any);
await ideMessenger.request("appAi/cancelVoiceRecording" as any, undefined as any);
```

Flutter 宿主侧由 `_AppAiVoiceHostAdapter` 接收这些消息，并分别调用：

```dart
Future<Map<String, Object?>> _startVoiceRecordingForGui()
Future<Map<String, Object?>> _stopVoiceRecordingForGui()
Future<Map<String, Object?>> _cancelVoiceRecordingForGui()
```

停止录音成功后，返回结构至少包含：

```dart
{
  'text': text,
  'audioUrl': audioUrl,
  'localPath': recordResult.path,
  'duration': recordResult.duration,
  'fileSize': recordResult.fileSize,
}
```

Web 侧拿到 `text` 后再插入输入框：

```ts
mainEditor?.chain().focus("end").insertContent(text.trim()).run();
```

## 本地录音工具

`AppAiVoiceRecordUtils` 使用 `record` 包的 `AudioRecorder`。

推荐设置：

```dart
await _recorder.start(
  const RecordConfig(
    encoder: AudioEncoder.aacLc,
    numChannels: 1,
    bitRate: 128000,
  ),
  path: path,
);
```

注意事项：

- 开始录音前先请求麦克风权限。
- 录音文件建议放在临时目录，文件名使用时间戳。
- 停止录音后返回本地路径、时长、文件大小。
- 取消录音时要删除临时文件。
- 组件销毁时要释放 `AudioRecorder`。

## OSS 上传要求

音频上传必须显式传 `DocType.audio`：

```dart
final audioUrl =
    await AliyunStoreManager.getInstance().uploadFileByFilePath(
  docType: DocType.audio,
  path: recordResult.path,
  fileExtensionEnum: _audioExtensionFromPath(recordResult.path),
  fileName: 'app_ai_voice_${DateTime.now().millisecondsSinceEpoch}',
);
```

不要使用默认 `FileExtension.jpg` 上传音频。推荐根据本地路径后缀映射：

```dart
FileExtension _audioExtensionFromPath(String path) {
  final extension = path.split('.').last.toLowerCase();
  switch (extension) {
    case 'mp3':
      return FileExtension.mp3;
    case 'wav':
      return FileExtension.wav;
    case 'aac':
      return FileExtension.aac;
    case 'amr':
      return FileExtension.amr;
    case 'm4a':
    default:
      return FileExtension.m4a;
  }
}
```

## 百炼 Paraformer 转写

当前使用阿里云百炼 Paraformer 录音文件识别 REST API。

提交任务：

```dart
final submitResponse = await http.post(
  Uri.parse(
    'https://dashscope.aliyuncs.com/api/v1/services/audio/asr/transcription',
  ),
  headers: <String, String>{
    'Authorization': 'Bearer $apiKey',
    'Content-Type': 'application/json',
    'X-DashScope-Async': 'enable',
  },
  body: jsonEncode(<String, Object?>{
    'model': 'paraformer-v2',
    'input': <String, Object?>{
      'file_urls': <String>[audioUrl],
    },
    'parameters': <String, Object?>{
      'language_hints': <String>['zh', 'en'],
    },
  }),
);
```

轮询任务：

```dart
final taskResponse = await http.post(
  Uri.parse('https://dashscope.aliyuncs.com/api/v1/tasks/$taskId'),
  headers: <String, String>{'Authorization': 'Bearer $apiKey'},
);
```

任务成功后从 `output.results[].transcription_url` 下载转写 JSON，再提取文本。

## 转写结果解析坑

百炼结果里可能同时包含：

- `transcripts[].text`：整段文本。
- `sentences[].text`：句子文本。
- `words[].text` / `tokens[].text` / `characters[].text`：字词级明细。

不要递归收集所有 `text` 字段，否则会出现整句后面又追加逐字换行的问题，例如：

```text
呃，你好你好你好，首先的话这个是什么？
呃
你
好
首
先
...
```

推荐解析优先级：

1. 优先取 `transcripts[].text`。
2. 没有整段文本时，再取 `sentences[].text`。
3. 兜底递归时必须跳过 `words`、`tokens`、`characters` 等字级明细。
4. 合并文本时保持顺序去重。

## 国际化与 AppAIPlugin 文案

新增录音转文字文案时，源头仍然放在 Flutter ARB：

- `lib/l10n/intl_*.arb`

然后生成：

- `lib/generated/*`

Flutter 外层通过 `getI18NKey(context).xxx` 取值，再把最终文案传给 AppAIPlugin Web 侧。不要在 TS 组件里维护第二套中文业务文案。

生成命令：

```bash
flutter pub global run intl_utils:generate
```

## 常见坑

- 不要让 AppAIPlugin 录音弹 Flutter 弹窗，HTML 输入区内联录音体验更一致。
- 不要把本地文件路径直接传给百炼，先上传 OSS 拿公网 URL。
- 不要漏传 `DocType.audio`。
- 不要用 `.jpg` 后缀上传音频。
- 不要递归拼接所有 `text` 字段，必须跳过字级明细。
- 不要只改 ARB 不生成 `lib/generated/*`，否则 macOS 编译会报 `getter isn't defined`。
- 不要在 `AliyunStoreManager.getInstance()` 创建单例时偷偷异步初始化 OSS，真正上传时再 `await init()` 更稳。
